# 🏥 MedAI - AI Medical Report Analyzer

MedAI is a web application built using **HTML, CSS, and Vanilla JavaScript**. It uses the **Google Gemini API** to analyze medical reports and provide easy-to-understand health insights .

## Features
- User Login & Register
- Upload PDF Medical Report
- AI Report Analysis
- AI Chat Assistant
- Report History
- Download PDF
- User Profile

## Technologies Used
- HTML
- CSS
- Vanilla JavaScript
- PDF.js
- jsPDF
- Google Gemini API

## Setup
1. Clone the repository.
2. Add your Gemini API key in `js/config.js`.
3. Run the project using Live Server.
